# GitSynth

GitSynth is an AI-powered Git Merge Conflict Resolution tool.

## Installation

```bash
npm install -g gitsynth
```

## Usage

After installation, you can use the `gitsynth` command globally:

```bash
gitsynth
```

## Platform Support

GitSynth includes binaries for:
- Linux (x64)
- macOS (x64)
- Windows (x64)

## Requirements

- Node.js 12 or later
